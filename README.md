# 临渊羡鱼 · 导航页

一个美观的个人导航页面，支持动态背景、一言API、深浅色模式切换，并带有完整的后台管理功能。

## 功能特性

- 🎨 美观的毛玻璃 UI 设计
- 🌓 深浅色模式切换
- 🖼️ 动态背景轮转
- 💬 一言 API 集成
- ⏱️ 网站运行时间统计
- 🔗 自定义导航链接
- 🔐 管理员后台（JWT 认证）
- ☁️ Cloudflare Pages + Workers 部署
- 💾 Cloudflare KV 存储

## 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **后端**: Cloudflare Workers (Hono.js)
- **存储**: Cloudflare KV
- **认证**: JWT (JSON Web Token)
- **部署**: Cloudflare Pages
